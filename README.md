# Mobile_Sales_PowerBI_Dashboard
The Mobile Sales Dashboard is an interactive business intelligence project developed using Microsoft Power BI.
This dashboard helps analyze mobile sales performance, customer behavior, payment methods, and regional sales trends through dynamic visualizations and KPIs.
✅ KPI Cards
Total Sales
Total Transactions
Average Sales
Total Quantity Sold
✅ Interactive Visualizations
Sales by City Map
Customer Rating Analysis
Sales Trend by Day
Quantity Trend Analysis
Payment Method Distribution
Mobile Model Performance
✅ Dynamic Filters / Slicers
Month Filter
City Filter

Payment Method Filter
