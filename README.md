# 📱 MOBILE SALES DASHBOARD

An interactive Power BI dashboard designed to analyze mobile sales performance, customer ratings, transactions, payment methods, and regional sales trends.

---

## 🚀 Project Overview

The **Mobile Sales Dashboard** provides valuable business insights through interactive visualizations and KPI metrics.  
This dashboard helps businesses monitor sales performance, customer satisfaction, payment behavior, and mobile model trends across different cities.


## 📌 Key Dashboard Insights

- 📈 Total Sales Performance
- 📦 Quantity Sold Analysis
- 💳 Payment Method Distribution
- 🌍 City-wise Sales Analysis
- ⭐ Customer Rating Insights
- 📅 Daily Sales Trend Analysis
